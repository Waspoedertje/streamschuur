# streamschuur
Give your Jellyfin server its own identity.

Powered by Jellyfin 

Streamschuur is an open-source branding and UI project for Jellyfin.

The goal is not to replace Jellyfin.

The goal is to make every self-hosted media server feel like its own streaming service.

Whether you’re sharing your server with your family, a few close friends, or just yourself, Streamschuur helps create a unique identity instead of another generic Jellyfin installation.

⸻

💡 Project Vision

This project started as a simple joke.

“What if a Jellyfin server didn’t feel like Jellyfin?”

After that, the idea quickly evolved.

Instead of creating another CSS theme, why not build something that allows every server owner to create their own streaming brand?

Examples:

🏚️ The Red Barn

🚀 GalaxyFlix

🎬 Cinema013

🎥 Grandpa’s Movie Vault

🌲 The Movie Cabin

All powered by Jellyfin.

All with their own identity.

🚫 What Streamschuur is NOT

Streamschuur is not:

* A Jellyfin replacement
* A media server
* A commercial product
* A piracy project
* Another Arr application

Streamschuur exists to improve the experience around Jellyfin while respecting the incredible work of the original project.
